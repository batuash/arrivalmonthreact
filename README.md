Arrival month
===========================

> An apllication to track working hours of employers 

Installing Dependencies
-----------------------

``` bash
npm install
```

Running Locally
---------------

``` bash
# serve with hot reload at localhost:8080
npm run start
```

Test
-----------------------

```bash
# run unit test
npm run test
```

[Live demo](https://batuash.github.io/arrivalmonthreact/)
